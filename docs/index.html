Introduction

P-E-BO is an advanced interactive desk assistant robot that combines emotional intelligence, real-time task management, and seamless intercommunication, powered by cutting-edge AI and IoT technologies.

Features

User Identification

Biometric recognition using AWS Rekognition for personalized user experiences.

Voice Interaction

Advanced NLP with Gemini API and Whisper for contextual voice responses.

Face Tracking

Dynamic face-tracking with servo-controlled pan-tilt for precise positioning.

Emotion Recognition

Real-time mood analysis via facial expressions and voice tone.

Touch Interaction

Haptic feedback sensors for intuitive gesture-based controls.

Expressive Movements

Arm and eye animations synchronized with emotional responses.

Solution Architecture

P-E-BO integrates modular hardware and AI-driven software for a seamless user experience.

  • Threaded architecture for concurrent face tracking, voice processing, and servo control.
  • Cloud integration with AWS Rekognition and S3 for real-time data processing.
  • Raspberry Pi-powered IoT with PCA9685 for precise motor control.
