Introduction

PEBO is an interactive desk assistant robot that blends emotional engagement, entertainment, task management, and intercommunication features.

Features

User Identification

Recognizes individual users via biometric data for personalized interactions.

AI Voice Response

Advanced natural language processing for contextual command understanding and execution.

Movement Tracking

Spatial awareness with automatic face-tracking for perfect positioning.

Emotion Recognition

Real-time mood analysis through voice tone and facial expression detection.

Affection Response

Responds to petting and gentle touches with happy lights, vibrations, and emotive sounds.

Touch Interaction

Responsive touch sensors with haptic feedback for precise gesture controls.

Solution Architecture

P-E-BO's architecture is designed to seamlessly integrate hardware and software components for optimal performance.

  • Modular hardware design for easy upgrades.
  • AI-powered software for voice recognition and task automation.
  • Cloud-based data storage for personalized settings and backups.
